# KeenKeeper - Keep Your Friendships Alive

A friendship tracking app that helps you never lose touch with the people who matter most.

## Technologies Used

- **React.js** - UI Library
- **Vite** - Build Tool
- **React Router DOM** - Navigation
- **Tailwind CSS** - Styling
- **Recharts** - Charts and Analytics
- **React Hot Toast** - Notifications
- **React Icons** - Icons

## Features

1. **Friend Management** - View all your friends in a grid layout, track days since last contact, and see status indicators (overdue, almost due, on-track)
2. **Interaction Timeline** - Log calls, texts, and video calls with friends. View your complete interaction history with filtering options
3. **Friendship Analytics** - Visual breakdown of your interaction types using a pie chart
